# vimfiles

Configuración personal para Vim 9.

_Nota: No sé si vaya a funcionar en tu sistema pues la configuración asume que usas Windows 10. En todo caso, tampoco apoyaré de manera técnica si lo quieres hacer._

## Comandos útiles

### Generar archivos de ayuda

```bash
vim -u NONE -c "helptags ALL" -c q
```
